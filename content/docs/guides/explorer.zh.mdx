# Explorer

Explorer 是 Dory 中用于理解数据库结构的对象浏览器，适合在编写查询前先建立结构认知。

## 当前方向

仓库中的发布说明将 Explorer 描述为一个统一的、驱动感知的界面，支持浏览如下对象：

- Database
- Schema
- Table
- View
- Materialized View
- Function
- Extension

## 适用场景

- 你刚接触一个数据库，需要快速建立结构认知。
- 你想在写 SQL 前先确认对象名称。
- 你需要验证当前连接是否暴露了预期的 Schema。

## 与其他能力的关系

Explorer 是后续工作流的上游。它帮助你验证 [数据库连接](/zh/docs/guides/connections)，提升 [SQL Console](/zh/docs/guides/sql-editor) 的编写效率，也让 [AI Chat](/zh/docs/guides/ai-chat) 的问题更具体。
