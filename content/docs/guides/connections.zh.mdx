# 数据库连接

连接是 Dory 所有后续能力的基础。Explorer、SQL 自动补全以及 AI 功能都依赖一个可用的数据库连接。

## 当前公开支持的数据库

公开仓库当前列出了：

- ClickHouse
- PostgreSQL
- MySQL
- MariaDB
- SQLite

DuckDB 仍处于计划中。

## 为什么连接很重要

连接建立后，Dory 可以利用实时 Schema 信息：

- 在编辑器中补全表、列、别名与函数。
- 让 AI 基于真实数据库结构生成或解释 SQL。
- 在 Explorer 中展示驱动感知的数据库对象视图。

## 建议流程

1. 为目标环境创建连接。
2. 确认当前数据库或 Schema 选择正确。
3. 打开 [Explorer](/zh/docs/guides/explorer) 验证对象是否可见。
4. 再进入 [SQL Console](/zh/docs/guides/sql-editor) 或 [AI Chat](/zh/docs/guides/ai-chat)。

## 需要注意的限制

不同数据库的功能深度并不完全一致。公开资料明确提到 ClickHouse 的集成深度高于其他驱动。
