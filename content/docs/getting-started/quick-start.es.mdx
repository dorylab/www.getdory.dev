# Inicio rápido

Esta página te ayuda a hacer que Dory funcione rápidamente. Elige el camino que coincida con tu objetivo.

## Opción 1: Pruebe la demostración en línea

Utilice la demostración en línea cuando desee explorar Dory sin instalar nada.

1. Abra [Demostración de Dory](https://app.getdory.dev).
2. Pruebe Explorer, SQL Console, AI Chat y gráficos con los datos de demostración disponibles.
3. Utilice la demostración para comprender el flujo de trabajo antes de conectar su propia base de datos.

## Opción 2: Instalar localmente

Utilice la instalación local cuando desee Dory en su propia máquina y necesite conectar bases de datos locales o de desarrollo.

```bash
brew install dory
```

Después de la instalación, abre Dory y crea tu primer espacio de trabajo.

## Opción 3: Ejecutar con Docker

Utilice Docker cuando desee un entorno autohospedado repetible o desee probar la configuración de implementación.

```bash
docker run -p 3000:3000 dorylab/dory
```

Para la implementación en producción, lea [Docker](/es/docs/deploy/docker) antes de exponer el servicio a un equipo.

## Siguiente paso

Después de que Dory se esté ejecutando, continúa con [Conecta tu base de datos](/es/docs/getting-started/connect-your-database).
