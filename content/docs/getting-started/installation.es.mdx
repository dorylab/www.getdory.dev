# Instalación

## Instalación de escritorio

### Homebrew en macOS

```bash
brew install dorylab/dory/dory
```

### Lanzar descargas

Utilice la página de lanzamientos de GitHub cuando desee una compilación de escritorio empaquetada para macOS o Windows:

- ` https://github.com/dorylab/dory/releases `

## Instalación de Docker

El repositorio público actualmente documenta este ejemplo de lanzamiento de Docker:

```bash
docker run -d --name dory \
  - p 3000:3000 \
  - e DS_SECRET_KEY="$(openssl rand -base64 32 | tr -d '\n')" \
  - e BETTER_AUTH_SECRET="$(openssl rand -hex 32)" \
  - e BETTER_AUTH_URL="http://localhost:3000" \
  - e DORY_AI_PROVIDER=openai \
  - e DORY_AI_MODEL=gpt-4o-mini \
  - e DORY_AI_API_KEY=your_api_key_here \
  - e DORY_AI_URL=https://api.openai.com/v1 \
  dorylab/dory:latest
```

## Notas de configuración mínima

- ` DS_SECRET_KEY `y ` BETTER_AUTH_SECRET `deben ser únicos por implementación.
- ` BETTER_AUTH_URL `debe coincidir con elURLpúblico que utilizas para acceder a Dory.
- Las variables de entorno relacionadas con la IA son necesarias si desea que funcionen las funciones de IA de Dory.
- Para obtener más orientación sobre la implementación, continúe con [Docker](/es/docs/deploy/docker) y [Self-Hosting](/es/docs/deploy/self-hosting).
