# 开始使用 Dory

Dory 是面向现代数据库的 AI 原生数据工作台。它将 SQL Console、数据库 Explorer、Schema 上下文感知 AI、查询结果分析与图表能力整合在同一工作空间中，帮助你更快完成从提问、查询到洞察的完整流程。

## Dory 适合谁

Dory 适用于需要频繁使用数据库的个人与团队，包括：

- 数据分析师：更快完成查询、验证假设与结果可视化。
- 数据工程师：高效浏览结构、调试 SQL、管理日常查询任务。
- 后端工程师：快速定位数据问题，理解业务数据结构。
- AI 团队与产品团队：通过自然语言降低 SQL 使用门槛。

## 为什么选择 Dory

- **AI 原生工作流**：基于当前连接、Schema 与上下文辅助生成 SQL、解释结果与优化查询。
- **一体化体验**：编辑器、Explorer、图表与查询记录保持在同一工作流中。
- **现代数据库支持**：面向 ClickHouse、PostgreSQL、MySQL、SQLite 等主流数据库场景设计。
- **灵活部署方式**：支持桌面端、本地开发环境、Docker 与团队部署。

## 核心能力一览

### SQL Console

现代 SQL 编辑器，支持编写、运行、结果查看与多轮迭代分析。

### AI Chat

结合数据库结构与当前上下文，生成 SQL、解释语句、辅助排查问题。

### Explorer

浏览数据库、Schema、表结构、字段与对象信息，快速理解数据模型。

### Charts & Results

将查询结果转换为图表，用于探索、汇报与分享。
