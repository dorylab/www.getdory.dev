# クイックスタート

このページは、Dory をすばやく実行するのに役立ちます。あなたの目標に合った道を選択してください。

## オプション 1: オンライン デモを試す

何もインストールせずに Dory を探索したい場合は、オンライン デモを使用してください。

1. [Dory デモ](https://app.getdory.dev) を開きます。
2. 利用可能なデモ データを使用して、エクスプローラー、SQL コンソール、AI チャット、およびグラフ作成を試します。
3. 独自のデータベースに接続する前に、デモを使用してワークフローを理解します。

## オプション 2: ローカルにインストールする

Dory を自分のマシンにインストールし、ローカル データベースまたは開発データベースに接続する必要がある場合は、ローカル インストールを使用します。

```bash
brew install dory
```

インストール後、Dory を開いて最初のワークスペースを作成します。

## オプション 3: Docker を使用して実行する

反復可能なセルフホスト環境が必要な場合、または展開設定をテストしたい場合は、Docker を使用します。

```bash
docker run -p 3000:3000 dorylab/dory
```

運用環境のデプロイメントについては、サービスをチームに公開する前に、[Docker](/ja/docs/deploy/docker) をお読みください。

## 次のステップ

Dory が実行されたら、[データベースに接続](/ja/docs/getting-started/connect-your-database) に進みます。
