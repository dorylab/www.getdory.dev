# Descripción general

La primera sesión de Dory tiene un camino corto: iniciar la aplicación, conectar una fuente de datos accesible, ejecutar una consulta limitada y revisar el resultado.

## Ruta de configuración

1. [Inicio rápido](/es/docs/getting-started/quick-start): elija una demostración alojada, una aplicación de escritorio o Docker.
2. [Instalación](/es/docs/getting-started/installation): instale la aplicación de escritorio o inicie un entorno de ejecución autohospedado.
3. [Conecte su base de datos](/es/docs/getting-started/connect-your-database): cree una conexión y confirme la visibilidad del esquema.
4. [Primera consulta en 3 minutos](/es/docs/getting-started/primera-consulta-en-3-minutos): ejecute una consulta limitada, revise los resultados y use IA para el siguiente paso.

## Consejos de producción mínima

- Utilice credenciales de solo lectura para bases de datos de producción.
- Confirmar el acceso a la red desde el tiempo de ejecución que ejecuta Dory, especialmente en Docker.
- Revise SQL generado por IA antes de ejecutarlo.
