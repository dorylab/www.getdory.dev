# 安装说明

## 桌面版安装

### macOS Homebrew

```bash
brew install dorylab/dory/dory
```

### Release 下载

如果你需要打包好的 macOS 或 Windows 桌面版，可以直接使用 GitHub Releases：

- `https://github.com/dorylab/dory/releases`

## Docker 安装

公开仓库当前提供了以下 Docker 启动示例：

```bash
docker run -d --name dory \
  -p 3000:3000 \
  -e DS_SECRET_KEY="$(openssl rand -base64 32 | tr -d '\n')" \
  -e BETTER_AUTH_SECRET="$(openssl rand -hex 32)" \
  -e BETTER_AUTH_URL="http://localhost:3000" \
  -e DORY_AI_PROVIDER=openai \
  -e DORY_AI_MODEL=gpt-4o-mini \
  -e DORY_AI_API_KEY=your_api_key_here \
  -e DORY_AI_URL=https://api.openai.com/v1 \
  dorylab/dory:latest
```

## 基本配置说明

- `DS_SECRET_KEY` 与 `BETTER_AUTH_SECRET` 应为每次部署单独生成。
- `BETTER_AUTH_URL` 需要与用户访问 Dory 的公开地址一致。
- 如需启用 AI 功能，必须配置对应的 AI 环境变量。
- 若你计划长期部署，请继续阅读 [Docker](/zh/docs/deploy/docker) 和 [自托管](/zh/docs/deploy/self-hosting)。
