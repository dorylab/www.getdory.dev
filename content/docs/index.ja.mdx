＃ いらっしゃいませ

Dory は、データベースの接続、スキーマの理解、SQL の作成、AI 支援の使用、結果のレビュー、再利用可能な分析の保存を行うための AI ネイティブのデータ ワークスペースです。

## 目標から始めましょう

| ゴール | エントリ |
| --- | --- |
| ドリーを初めて走らせる | [はじめに](/ja/docs/getting-started) |
| 特定の種類のデータベースに接続する | [データベースサポート](/ja/docs/database-support) |
| SQL コンソール、AI チャット、エクスプローラー、チャート、保存されたクエリについて学ぶ | [コア機能](/ja/docs/core-features) |
| 一般的な分析タスクをエンドツーエンドで完了する | [ワークフロー](/ja/docs/workflows) |
| チームまたはサーバー環境に展開する | [デプロイ](/ja/docs/deploy) |
| プロバイダー、ショートカット、MCP、設定、またはトラブルシューティングを構成する | [参考資料](/ja/docs/reference) |

## 共通エントリ

- [はじめに](/ja/docs/getting-started): データベースをインストール、起動、接続し、最初のクエリを実行します。
- [コア機能](/ja/docs/core-features):SQL コンソール、AI チャット、エクスプローラー、チャート、接続、保存されたクエリ。
- [ワークフロー](/ja/docs/workflows): 未知のテーブルの探索、SQL の生成、遅いクエリのデバッグ、グラフの構築、分析の再利用。
- [リファレンス](/ja/docs/reference): AI プロバイダー、ショートカット、SQL スニペット、設定、MCP、およびFAQ。
