## 快速答案

- Dory AI 提供商通常由 `DORY_AI_PROVIDER`、`DORY_AI_MODEL`、`DORY_AI_API_KEY` 和必要时的 `DORY_AI_URL` 控制。
- SQL 编辑器支持运行、格式化、保存、内联 AI、新建 Tab 和结果搜索等快捷键。
- 查询预览默认应保持较小返回行数；只有分析确实需要时再提高限制。
- MCP token 与数据库凭据分离，可以按客户端单独撤销。
- Docker 部署读取本地 SQLite 或 DuckDB 文件时，需要挂载包含这些文件的宿主机目录。
