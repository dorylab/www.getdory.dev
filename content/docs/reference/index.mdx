## Quick answers

- Dory AI provider selection is controlled by `DORY_AI_PROVIDER`, `DORY_AI_MODEL`, `DORY_AI_API_KEY`, and, when required, `DORY_AI_URL`.
- The SQL editor supports run, format, save, inline AI, new tab, and result search shortcuts.
- Query result previews should stay limited by default; raise limits only when the analysis needs more rows.
- MCP tokens are separate from database credentials and can be revoked per client.
- Docker deployments that read local SQLite or DuckDB files should mount the host directory that contains those files.
