# Dory 文档

Dory 是面向现代数据库的 AI 原生数据工作台。它将 Schema 感知的 SQL Console、对话式 AI、数据库 Explorer、查询结果分析与图表能力整合在同一工作空间中，帮助你更快完成从提问到执行再到洞察的完整流程。

## 5 分钟快速开始

1. 安装 Dory（桌面版、Homebrew 或 Docker）。
2. 连接 ClickHouse、PostgreSQL、MySQL 等数据库。
3. 用自然语言生成 SQL，或直接在编辑器中编写查询。
4. 查看结果、转换图表，并保存常用查询。

## 你可以用 Dory 做什么

- 使用 AI 生成 SQL、解释语句，并结合上下文完成数据探索。
- 在现代 SQL Console 中编写、运行与管理查询。
- 浏览数据库结构、表字段与对象关系。
- 将查询结果转换为图表，用于分析与分享。
- 保存常用查询与工作流，沉淀团队知识。

## 支持的数据库与 AI 提供商

Dory 当前重点支持以下生态：

- 数据库：ClickHouse、PostgreSQL、Neon、MySQL、MariaDB、SQLite、DuckDB。
- AI 提供商：OpenAI、Anthropic、Google、Qwen、xAI，以及兼容 OpenAI API 的模型服务。

## 为什么选择 Dory

- **AI 原生体验**：不是外挂聊天框，而是深度理解数据库上下文的工作流助手。
- **一体化工作空间**：编辑器、Explorer、结果分析与图表无需频繁切换工具。
- **现代部署方式**：支持桌面端、本地运行、Docker 与团队部署场景。
