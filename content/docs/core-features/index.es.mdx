# Descripción general

El espacio de trabajo principal de Dory incluye consola SQL, chat AI, explorador, gráficos de resultados, administración de conexiones y consultas guardadas.

## Mapa de características

| Característica | Úselo para |
| --- | --- |
| [Consola SQL](/es/docs/core-features/sql-console) | Escriba, ejecute, depure y guarde SQL con el contexto del esquema. |
| [Chat AI](/es/docs/core-features/ai-chat) | Genere SQL, explique consultas, corrija errores y continúe el análisis con el contexto de la base de datos. |
| [Explorador](/es/docs/core-features/explorer) | Inspeccione bases de datos, esquemas, tablas, columnas, filas de muestra y detalles de objetos. |
| [Gráficos y resultados](/es/docs/core-features/charts-results) | Revise las tablas de resultados y convierta los resultados SQL listos para gráficos en resúmenes visuales. |
| [Consultas guardadas](/es/docs/core-features/consultas-guardadas) | Guarde SQL útiles y análisis reutilizables para usted o su equipo. |
| [Conexiones](/es/docs/core-features/conexiones) | Administre la configuración de conexión de la base de datos y las rutas de archivos locales. |
| [Monitoreo](/es/docs/core-features/monitoreo) | Inspeccione la actividad de consultas de ClickHouse, consultas lentas, errores y señales operativas. |
| [Privilegios](/es/docs/core-features/privilegios) | Administre usuarios, roles, permisos y concesiones de clústeres de ClickHouse. |

## Tareas comunes

- Explorar una tabla desconocida: [Explorar tablas desconocidas](/es/docs/workflows/explore-unknown-tables)
- Borrador SQL con IA: [Generar SQL](/es/docs/workflows/generate-sql-with-ai)
- Crear un gráfico a partir de los resultados: [Crear gráficos a partir de SQL](/es/docs/workflows/build-charts-from-sql)
