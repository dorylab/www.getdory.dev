＃ 概要

Dory のメイン ワークスペースには、SQL コンソール、AI チャット、エクスプローラー、結果グラフ、接続管理、および保存されたクエリが含まれています。

## 機能マップ

| 特徴 | 用途: |
| --- | --- |
| [SQL コンソール](/ja/docs/core-features/sql-console) | スキーマ コンテキストを使用して SQL を書き込み、実行、デバッグ、保存します。 |
| [AI チャット](/ja/docs/core-features/ai-chat) | SQL を生成し、クエリを説明し、エラーを修正し、データベース コンテキストで分析を続行します。 |
| [エクスプローラー](/ja/docs/core-features/explorer) | データベース、スキーマ、テーブル、列、サンプル行、オブジェクトの詳細を検査します。 |
| [チャートと結果](/ja/docs/core-features/charts-results) | 結果テーブルを確認し、グラフに対応した SQL 出力を視覚的な概要に変換します。 |
| [保存されたクエリ](/ja/docs/core-features/saved-queries) | 自分自身またはチームのために有用な SQL と再利用可能な分析を保存します。 |
| [接続](/ja/docs/core-features/connections) | データベース接続設定とローカル ファイル パスを管理します。 |
| [モニタリング](/ja/docs/core-features/monitoring) | ClickHouse クエリ アクティビティ、遅いクエリ、エラー、および操作上の信号を検査します。 |
| [権限](/ja/docs/core-features/privileges) | ClickHouse ユーザー、ロール、権限、クラスター対応の付与を管理します。 |

## 一般的なタスク

- 未知のテーブルを探索する: [Explore Unknown Tables](/ja/docs/workflows/explore-unknown-tables)
- AI を使用した SQL のドラフト: [SQL の生成](/ja/docs/workflows/generate-sql-with-ai)
- 結果からグラフを作成: [Build Charts from SQL](/ja/docs/workflows/build-charts-from-sql)
