# Consultas guardadas

Las consultas guardadas convierten SQL útil en análisis reutilizables. Son mejores para consultas que espera ejecutar nuevamente o compartir con compañeros de equipo.

## Lo que puedes ahorrar

- Informes reutilizables.
- Controles operativos.
- Depuración de consultas.
- Listo para gráficos SQL.
- Consultas generadas o mejoradas con IA tras revisión manual.

## Cuándo guardar

- La misma pregunta surge repetidamente.
- La consulta contiene una lógica empresarial no obvia.
- Es posible que un compañero de equipo necesite reutilizarlo.
- El resultado impulsa un gráfico o flujo de trabajo recurrente.

## Recomendaciones de nombres

Utilice nombres que describan la pregunta comercial, no solo el nombre de la tabla.

Buenos ejemplos:

- Usuarios activos semanales por área de producto.
- Pagos fallidos en las últimas 24 horas.
- Consultas lentas de ClickHouse por usuario.

## Flujo de uso

1. Escriba y valide SQL.
2. Limpie el formato y elimine los filtros temporales.
3. Agregue un nombre y una descripción claros.
4. Guarde la consulta.
5. Revíselo periódicamente para mantener actualizadas las suposiciones.

## Práctica en equipo

Documente rangos de tiempo, filtros, uniones y definiciones de métricas para que otros puedan reutilizar la consulta de forma segura.
