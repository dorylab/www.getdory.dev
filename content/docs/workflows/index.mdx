# Dory Workflow Guides

Workflows document the most common analysis tasks in Dory. Each guide connects Connections, Explorer, SQL Console, AI Chat, Charts & Results, and Saved Queries into a repeatable process.

## Recommended Workflows

| Workflow | Use case | Main features |
| --- | --- | --- |
| [Explore Unknown Tables](/en/docs/workflows/explore-unknown-tables) | Understand unfamiliar schemas | Explorer, table preview, AI Chat |
| [Generate SQL with AI](/en/docs/workflows/generate-sql-with-ai) | Turn business questions into SQL | AI Chat, SQL Console, schema context |
| [Debug Slow Queries](/en/docs/workflows/debug-slow-queries) | Investigate expensive SQL | SQL Console, AI Chat, ClickHouse Monitoring |
| [Build Charts from SQL](/en/docs/workflows/build-charts-from-sql) | Shape query output for visualization | SQL Console, Charts & Results, AI |
| [Save & Reuse Analysis](/en/docs/workflows/save-reuse-analysis) | Preserve useful queries and team knowledge | Saved Queries, folders, descriptions |

## Suggested Learning Path

1. Explore schema with [Explore Unknown Tables](/en/docs/workflows/explore-unknown-tables).
2. Generate a first query with [Generate SQL with AI](/en/docs/workflows/generate-sql-with-ai).
3. Run and refine the query in [SQL Console](/en/docs/core-features/sql-console).
4. Debug performance with [Debug Slow Queries](/en/docs/workflows/debug-slow-queries).
5. Visualize stable output with [Build Charts from SQL](/en/docs/workflows/build-charts-from-sql).
6. Save reusable work with [Save & Reuse Analysis](/en/docs/workflows/save-reuse-analysis).

## Quick Answer

Dory workflows help you move from unfamiliar data to reviewed SQL, chart-ready results, reusable analysis, and slow-query debugging without leaving the same workspace.
