# Descripción general

Los flujos de trabajo documentan las tareas de análisis más comunes en Dory. Cada guía conecta Conexiones, Explorador, Consola SQL, Chat AI, Gráficos y resultados y Consultas guardadas en un proceso repetible.

## Flujos de trabajo recomendados

| Flujo de trabajo | Caso de uso | Características principales |
| --- | --- | --- |
| [Explorar tablas desconocidas](/es/docs/workflows/explorar-tablas-desconocidas) | Comprender esquemas desconocidos | Explorador, vista previa de tabla, chat AI |
| [Generar SQL con IA](/es/docs/workflows/generate-sql-with-ai) | Convierta las preguntas comerciales en SQL | AI Chat, consola SQL, contexto de esquema |
| [Depurar consultas lentas](/es/docs/workflows/debug-slow-queries) | Investigar caro SQL | Consola SQL, Chat AI, Monitoreo ClickHouse |
| [Crear gráficos a partir de SQL](/es/docs/workflows/build-charts-from-sql) | Salida de consulta de forma para visualización | Consola SQL, gráficos y resultados, IA |
| [Análisis de guardar y reutilizar](/es/docs/workflows/save-reuse-analysis) | Preservar consultas útiles y conocimiento del equipo | Consultas guardadas, carpetas, descripciones. |

## Ruta de aprendizaje sugerida

1. Explore el esquema con [Explorar tablas desconocidas](/es/docs/workflows/explore-unknown-tables).
2. Genere una primera consulta con [Generar SQL con AI](/es/docs/workflows/generate-sql-with-ai).
3. Ejecute y refine la consulta en [SQL Console](/es/docs/core-features/sql-console).
4. Depure el rendimiento con [Depurar consultas lentas](/es/docs/workflows/debug-slow-queries).
5. Visualice una salida estable con [Crear gráficos a partir de SQL](/es/docs/workflows/build-charts-from-sql).
6. Guarde el trabajo reutilizable con [Guardar y reutilizar análisis](/es/docs/workflows/save-reuse-analysis).
