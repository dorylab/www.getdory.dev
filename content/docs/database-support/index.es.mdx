# Descripción general

Dory es un espacio de trabajo de datos nativo de IA para bases de datos modernas. Puede conectar bases de datos compatibles, explorar esquemas, escribir SQL, generar consultas con IA, inspeccionar resultados, crear gráficos y guardar consultas reutilizables.

## Flujo de trabajo típico de la base de datos de Dory

1. Cree una conexión en [Conexiones](/es/docs/core-features/connections).
2. Explore bases de datos, esquemas, tablas y columnas en [Explorador](/es/docs/core-features/explorer).
3. Escriba y ejecute SQL en [consola SQL](/es/docs/core-features/sql-console).
4. Utilice [Chat AI](/es/docs/core-features/ai-chat) para generar, explicar, corregir u optimizar consultas.
5. Valide los resultados y cree gráficos en [Gráficos y resultados](/es/docs/core-features/charts-results).
6. Guarde SQL reutilizable en [Consultas guardadas](/es/docs/core-features/saved-queries).

## Elija un método de conexión

- Utilice la aplicación de escritorio para desarrollo local y acceso directo desde su máquina.
- Utilice Docker o autohospedaje cuando un equipo necesite acceso compartido.
- Utilice rutas de archivos para SQLite y DuckDB solo cuando el tiempo de ejecución de Dory pueda leer esos archivos.
- Utilice la configuración de red de nube o túnelSSHcuando las bases de datos no sean accesibles públicamente.
