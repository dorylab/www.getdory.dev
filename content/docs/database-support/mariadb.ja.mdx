# マリアDB

<img src="/icons/databases/mariadb.svg" alt="MariaDB logo" width="48" height="48" />

Dory は、Web アプリケーション、内部システム、レポート、運用分析のための専用の接続タイプとして MariaDB をサポートします。 MariaDB は Dory の MySQL に似ていますが、バージョン固有の SQL の動作は異なる場合があります。

## 最適な使用例

- ユーザー、注文、製品、支払い、その他のアプリケーション テーブルをクエリします。
- なじみのない MariaDB スキーマを調べます。
- AI を使用してレポート SQL を生成します。
- 実稼働データまたはステージング データの問題を調査します。
- チーム用に再利用可能な MariaDB クエリを保存します。

## サポートされる機能

| 能力 | MariaDB のサポート |
| --- | --- |
| 接続 | ホスト、ポート、データベース、ユーザー名、パスワード、SSL、およびSSHトンネル。 |
| エクスプローラ | データベース、テーブル、ビュー、列。 |
| SQL コンソール | MariaDB SQL の編集、実行、および結果。 |
| AI チャット | スキーマに基づいた SQL の生成、説明、および修正。 |
| チャートと結果 | 集計結果の視覚的分析。 |
| 保存されたクエリ | 再利用可能な MariaDB SQL を保存します。 |

## ヒント

特に JSON 関数、ウィンドウ関数、またはシステム テーブルが重要な場合、ターゲット データベースが MariaDB であることを AI に伝えます。運用環境では読み取り専用アカウントと明示的な時間範囲を使用します。

## 接続要件

- ホスト、ポート、データベース名、ユーザー名、およびパスワード。
- Dory ランタイムからのネットワーク アクセス。
- 展開で必要な場合は、SSL設定。
- 対象のスキーマおよびテーブルを読み取る権限を持つユーザー。

## FAQ

### MySQL ページの推奨事項を MariaDB に使用できますか?

はい、ほとんどの接続とクエリの実践は重複します。関数、JSON 機能、および SQL モードについては、引き続き MariaDB 固有のバージョンの動作を確認してください。

### 接続に失敗した場合、まず何を確認すればよいですか?

ホスト、ポート、資格情報、ファイアウォール ルール、データベース名、およびユーザーがターゲット スキーマにアクセスできるかどうかを確認します。
