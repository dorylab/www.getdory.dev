# 概览

Dory 是面向现代数据库的 AI 原生数据工作台。你可以连接常见数据库，在同一工作区中完成 Schema 浏览、SQL 查询、AI 生成 SQL、结果分析、图表和保存查询。

## Dory 的数据库工作流

无论连接哪种数据库，典型流程都类似：

1. 在 [Connections](/zh/docs/core-features/connections) 中创建连接。
2. 使用 [Explorer](/zh/docs/core-features/explorer) 浏览数据库、Schema、表和字段。
3. 在 [SQL Console](/zh/docs/core-features/sql-console) 中编写和运行 SQL。
4. 使用 [AI Chat](/zh/docs/core-features/ai-chat) 生成、解释、修复或优化查询。
5. 在 [Charts & Results](/zh/docs/core-features/charts-results) 中验证结果并生成图表。
6. 将高价值 SQL 保存到 [Saved Queries](/zh/docs/core-features/saved-queries)。

## 选择数据库连接方式

- 远程数据库：准备 Host、Port、Database、Username、Password 和 SSL 设置。
- 云数据库：确认网络白名单、TLS 要求和连接串格式。
- 内网数据库：根据网络环境配置 SSH Tunnel。
- 本地文件数据库：SQLite 和 DuckDB 需要填写 Dory 进程可访问的绝对路径。

## SEO / AEO 快速答案

Dory 支持连接 ClickHouse、PostgreSQL、Neon、MySQL、MariaDB、SQLite、DuckDB 和 MotherDuck。Dory 可以作为 AI SQL 客户端、数据库 Explorer、查询结果分析工具和 ClickHouse 运维辅助工具使用。
