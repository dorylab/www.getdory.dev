# MySQL

<img src="/icons/databases/mysql.svg" alt="MySQL logo" width="48" height="48" />

Dory は、Web アプリケーション データベース、運用分析、レポート作成、運用トラブルシューティングのために MySQL と MariaDB をサポートしています。SQL コンソール、エクスプローラー、AI チャット、結果グラフ、保存されたクエリを 1 つのワークフローで使用します。

## 最適な使用例

- ユーザー、注文、製品、支払い、その他のアプリケーション テーブルをクエリします。
- なじみのないアプリケーション スキーマを理解する。
- AI を使用してレポート SQL またはトラブルシューティング SQL を生成します。
- 繰り返し発生する操作クエリを保存します。
- 集計されたクエリ結果を視覚化します。

## サポートされる機能

| 能力 | MySQL / MariaDB のサポート |
| --- | --- |
| 接続 | ホスト、ポート、データベース、ユーザー名、パスワード、SSL、およびSSHトンネル。 |
| エクスプローラ | データベース、テーブル、ビュー、列。 |
| SQL コンソール | MySQL SQL の編集、実行、および結果。 |
| AI チャット | スキーマに基づいた SQL の生成、説明、およびエラー修正。 |
| チャートと結果 | 集計結果の視覚的分析。 |
| 保存されたクエリ | 再利用可能な MySQL クエリを保存します。 |

## クエリのヒント

探索中に`LIMIT `を使用し、明示的な時間フィルターを追加し、AI 生成の結合を実行する前にインデックスをチェックし、バージョン固有の関数サポートを考慮します。

## 接続要件

- ホスト、ポート、データベース名、ユーザー名、およびパスワード。
- Dory ランタイムからのネットワーク アクセス。
- サーバーまたはクラウドプロバイダーが必要とする場合は、SSL設定。
- スキーマメタデータとターゲットテーブルを読み取る権限。

## FAQ

### Dory は MariaDB をサポートしていますか?

はい。 MariaDB には独自の接続タイプがあり、MySQL と同様に機能します。

### MySQL 接続が失敗するのはなぜですか?

ホスト、ポート、データベース、ユーザー名、パスワード、ネットワーク許可リスト、およびSSL要件を確認してください。

### AI が生成した MySQL SQL が実際の構文と一致しない場合はどうなりますか?

AI に MySQL または MariaDB のバージョンを伝え、正確なエラーを貼り付けます。再実行する前に、関数、日付構文、および JSON 操作を確認してください。
