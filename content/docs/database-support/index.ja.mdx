＃ 概要

Dory は、最新のデータベース用の AI ネイティブ データ ワークスペースです。サポートされているデータベースに接続し、スキーマを参照し、SQL を書き込み、AI を使用してクエリを生成し、結果を検査し、グラフを作成し、再利用可能なクエリを保存することができます。

## 一般的な Dory データベースのワークフロー

1. [接続](/ja/docs/core-features/connections) に接続を作成します。
2. [エクスプローラー](/ja/docs/core-features/explorer) でデータベース、スキーマ、テーブル、列を参照します。
3. [SQL Console](/ja/docs/core-features/sql-console) で SQL を記述して実行します。
4. [AI Chat](/ja/docs/core-features/ai-chat) を使用して、クエリを生成、説明、修正、または最適化します。
5. 結果を検証し、[グラフと結果](/ja/docs/core-features/charts-results) でグラフを作成します。
6. 再利用可能な SQL を [保存されたクエリ](/ja/docs/core-features/saved-queries) に保存します。

## 接続方法を選択します

- ローカル開発およびマシンからの直接アクセスにはデスクトップ アプリを使用します。
- チームが共有アクセスを必要とする場合は、Docker またはセルフホスティングを使用します。
- SQLite および DuckDB のファイル パスは、Dory ランタイムがこれらのファイルを読み取れる場合にのみ使用します。
- データベースがパブリックにアクセスできない場合は、SSHトンネルまたはクラウド ネットワーク設定を使用します。
