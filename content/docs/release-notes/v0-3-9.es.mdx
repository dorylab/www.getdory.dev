# v0.3.9

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.3.9.md)

## ¿Qué ha cambiado?
### Correcciones
- Se corrigieron cuadros de diálogo de error duplicados que aparecían cuando fallaban las comprobaciones de actualización; ahora solo se muestra un cuadro de diálogo por falla, lo que mejora la experiencia del usuario durante los errores del actualizador.
