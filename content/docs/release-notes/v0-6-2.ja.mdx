# v0.6.2

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.6.2.md)

## 何が変わったのか
### 国際化とローカリゼーション
- デスクトップ アプリと Web アプリの両方で、日本語 (ja)、スペイン語 (es)、および簡体字中国語 (zh_CN) の完全サポートを追加しました。
- アクセシビリティと永続性が向上した更新された言語選択 UI
- コアアプリ領域のローカライズされた文字列が含まれています:SQL コンソールの結果、設定パネル (組織、請求、外観)、サイドバーのナビゲーション、およびバージョン情報パネル
- サポートされているすべての言語で GitHub クイックスタート ガイド コンテンツを追加しました
### ドキュメント
- `README.md `を更新し、セットアップ手順とプロジェクトの概要を更新しました
