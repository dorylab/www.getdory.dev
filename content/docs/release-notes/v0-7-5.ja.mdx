# v0.7.5

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.7.5.md)

## 何が変わったのか
### 修正
- チャットボットでのデータベース スキーマの提案とセレクターの動作が修正され、テーブルまたは列を選択する際の精度と応答性が向上しました。
- チャットボットでの提案されたプロンプト ストリーミングとセッションのハンドオフに関する問題が解決され、ユーザー プロンプトと AI が生成した提案の間のスムーズな移行が保証されます。
- チャットボット コンポーネントおよび関連リソースから不要なデバッグ ログを削除し、ノイズを削減し、実行時のパフォーマンスを向上させました。
### 改善点
- 主要なチャットAPIエンドポイント (`/api/chat `、`/api/chat/sql-runner `) および SQL ツール ロジックに` performanceGuard `を追加し、遅いクエリやリソースを大量に消費するクエリによる応答性の低下を防ぎます。
### 依存関係
- ` next-intl `を v4.9.0 から v4.9.1 に更新しました。
- Next.js を v16.2.2 から v16.2.3 に更新しました。
