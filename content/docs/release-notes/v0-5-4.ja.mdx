# v0.5.4

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.5.4.md)

## 何が変わったのか
＃＃＃ 特徴
- SQL コンソール結果テーブルに自動列幅調整 (autofit) が追加され、手動でサイズを変更しなくてもクエリ結果が読みやすくなりました。
- 結果テーブルの応答性とコンテンツに合わせた動作を向上させるために、動的な列幅の計算が導入されました。
### 修正
- 結果テーブルの`<thead>`の水平スクロール (` scroll-x `) の問題を修正し、スクロール中にヘッダーがデータ行と適切に位置合わせされた状態を維持できるようにしました。
### 改善点
- 視覚的な一貫性と UI の明瞭さの向上のために、結果テーブルのセクションの背景スタイルを更新しました。
