# v0.12.0

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.12.0.md)

## ¿Qué ha cambiado?
### Características
- **Se agregó compatibilidad con DuckDB**: los usuarios ahora pueden conectarse y consultar bases de datos DuckDB directamente desde la aplicación web, incluida la configuración de la conexión, la ejecución de SQL, la exploración de tablas y la introspección de esquemas.
### Dependencias
- Actualizado `fast-xml-builder ` de `1.1.4` a `1.2.0`.
- Actualizado ` fast-uri `de `3.1.0` a `3.1.2`.
