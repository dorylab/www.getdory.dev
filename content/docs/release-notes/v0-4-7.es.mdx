# v0.4.7

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.4.7.md)

## ¿Qué ha cambiado?
### Correcciones
- Se corrigieron fallas de verificación de correo electrónico en el flujo de autenticación de demostración.
- Se resolvieron errores relacionados con la integración de `better-auth `, incluidas actualizaciones de esquema y migración para PostgreSQL y pglite.
### Características
- Se agregaron verificaciones de actualización periódicas a la aplicación de escritorio para mejorar la confiabilidad de la actualización automática.
### Mejoras
- Se actualizó Next.js de v14.2.35 a v16.2.0 en todas las aplicaciones web, lo que permite componentes de servidor React modernos, enrutamiento mejorado y mejoras de rendimiento.
- Se actualizó KyselyORMde v0.28.12 a v0.28.14 para mejoras menores de estabilidad y seguridad de tipos.
