# v0.26.0

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.26.0.md)

## What's Changed

### Features
- Added full Snowflake connection support, including dedicated driver, form fields, and UI integration across the web app.

### Improvements
- Enhanced workspace navigation and connection handling for smoother user flow when managing multiple connections.
- Improved SQL panel resizing behavior for better usability in the SQL console.

### Fixes
- Fixed inconsistent SQL panel resizing that affected layout stability during interaction.

### Internal
- Made connection payload validation stricter to improve data integrity and security.
- Added end-to-end test coverage for the SQL console workbench.
