# v0.6.2

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.6.2.md)

## ¿Qué ha cambiado?
### Internacionalización y localización
- Se agregó soporte completo para japonés (ja), español (es) y chino simplificado (zh_CN) en aplicaciones web y de escritorio.
- Interfaz de usuario de selección de idioma actualizada con accesibilidad y persistencia mejoradas
- Se incluyen cadenas localizadas para áreas principales de la aplicación: resultados de la consola SQL, paneles de configuración (organización, facturación, apariencia), barra lateral de navegación y el panel Acerca de
- Se agregó contenido de la guía de inicio rápido de GitHub en todos los idiomas admitidos.
### Documentación
- `README.md ` actualizado con instrucciones de configuración actualizadas y descripción general del proyecto
