# v0.6.0

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.6.0.md)

## 何が変わったのか
＃＃＃ 特徴
- アプリケーション全体に次のような実験的な SQLite サポートを追加しました。
  - SQLite データベース用の SQL コンソールおよびテーブル ブラウザとの統合
  - SQLite クエリ支援用に調整された新しい Copilot コンテキスト タイプ
  - SQLite バイナリを含むようにデスクトップ アプリのパッケージ化と CI ワークフローを更新しました
＃＃＃ メンテナンス
- 間接的な依存関係`brace-expansion `を`1.1.12`から`1.1.13`に更新しました (セキュリティと安定性のパッチ)
