# v0.4.2

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.4.2.md)

## 何が変わったのか
### 修正
- 権限管理ページ (ロール、ユーザー、およびメイン権限ビュー) でのビルドの失敗を修正しました。
- PostgreSQL 接続の UI で **Monitoring** セクションと **Privileges** セクションを非表示にしました。これらの機能はサポートされていません。
