# v0.12.1

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.12.1.md)

## 何が変わったのか
＃＃＃ 特徴
- **DuckDB サポートの追加**: ユーザーは、接続セットアップ、SQL コンソールの統合、テーブルの参照、スキーマ イントロスペクションなど、Web アプリから直接 DuckDB データベースに接続してクエリを実行できるようになりました。
### 修正
- **DuckDB ランタイム エラーを修正**: 安定した DuckDB 接続とクエリ実行を確保するために、デスクトップ アプリの初期化と実行の問題が解決されました。
- **ベータ リリース ワークフローを修正**: 適切なベータ ビルドを妨げていた`beta.yml ` GitHub Actions ワークフローの構成エラーを修正しました。
