# v0.5.0

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.5.0.md)

## ¿Qué ha cambiado?
### Nueva compatibilidad con bases de datos
- Se agregó soporte básico de conexión MySQL, incluido el formulario de conexión, exploración de metadatos, recuperación de información de tablas y manejo del dialecto SQL.
- Se agregó soporte de conexión MariaDB, reutilizando gran parte de la infraestructura MySQL y al mismo tiempo proporcionando capacidades específicas de MariaDB y componentes del explorador.
### Mejoras en la interfaz de usuario
- Se eliminó el filtro de escala de grises de los avatares de los usuarios en la barra lateral de la aplicación para mejorar la coherencia visual y la accesibilidad.
