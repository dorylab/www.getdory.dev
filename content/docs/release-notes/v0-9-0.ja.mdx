# v0.9.0

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.9.0.md)

## 何が変わったのか
＃＃＃ 特徴
- **AI 使用状況の追跡**: スキーマの説明、テーブルの概要、チャットのやり取りなど、AI 機能の使用状況を監視するための新しい AI 使用状況ダッシュボードを追加しました。
- **Pro プランとデスクトップ サブスクリプション**: 機能が強化された Pro プランが導入され、請求設定でのデスクトップ サブスクリプション管理のサポートが追加されました。
- **請求ページの見直し**: プランのアップグレード、デスクトップのサブスクリプション、ロケール間でのより明確なプランの比較をサポートするために請求ページを再設計しました。
### 改善点
- **サイドバーとナビゲーションの更新**: アプリのサイドバーが更新され、構成が改善され、プラン インジケーターが更新され、ナビゲーションが合理化されました。その後、UI がすっきりするようにプラン タグを非表示にするように改良されました。
- **ローカリゼーション**: すべての新しい請求機能と AI 機能をサポートするために、英語、スペイン語、日本語、中国語の翻訳を拡張および更新しました。
### 修正
- **E2Eテストの安定性**: 特に SQL コンソール フローと保存されたクエリ フォルダー周りの不安定なエンドツーエンド テストが修正され、CI の信頼性が向上しました。
