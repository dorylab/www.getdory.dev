# v0.10.0

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.10.0.md)

## 何が変わったのか
### AI を活用した結果の洞察
- **SQL 結果セットの AI プロファイリング**を導入し、説明的な概要、統計的洞察、実用的な推奨事項を SQL コンソールで直接自動的に生成します。
- 専用の **概要タブ** と **結果セット インサイト パネル** を結果テーブル UI に追加し、データ分布、外れ値、相関関係、潜在的な異常などの AI 生成のコンテキストを明らかにしました。
- **AI を活用した分析アクション** を開始しました。これには、クイック変換 (例: 「トップ 10 のフィルター」、「カテゴリによるピボット」) や自然言語駆動のチャートの提案が含まれます。これは、コパイロット パネルと結果の概要からアクセスできます。
### UX とパフォーマンスの改善
- **安定したレイアウト動作**: 表/グラフ/概要タブを切り替えるときのちらつきと位置のずれを修正しました。ヘッダーとチャートが一貫した位置を維持するようになりました。
- **最適化された結果キャッシュ**: メタデータのみの更新 (列名の変更など) 中の結果キャッシュの不必要な無効化を防止し、クエリの再実行速度を向上させます。
- **洗練されたチャート ツール**: 結果プロファイル全体で自動チャート生成ロジックが統合され、バケット化とアクション グループ化が改善されてチャート コントロール バーが更新され、キャンバスの応答性が向上しました。
### 修正
- 対話型並べ替え中に UI がフリーズする原因となった、仮想化された結果テーブルでの無限並べ替え更新ループを解決しました。
- 重複した AI Insight のプリフェッチを排除し、読み込みタイミングを調整して、結果をレンダリングする際の待ち時間と冗長なAPI呼び出しを削減しました。
