# v0.7.10

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.7.10.md)

## 何が変わったのか
### 認証
- ゲスト ログイン機能を削除し、新しいデモ サインイン フローを導入しました。これにより、ユーザーはアカウントを作成せずにアプリケーションをすばやく探索できるようになります。
### SQL コンソール UX の改善
- 洗練された SQL モードの動作、よりスマートなタブ管理、および改善されたエディターの応答性により、SQL コンソールでの Copilot エクスペリエンスが強化されました。
- 読みやすさ、対話性、アクセシビリティを向上させるために、結果テーブル コンポーネントを更新しました。
- 構文を意識した提案やエラーの強調表示など、SQL 編集をよりスムーズにするための洗練された Monaco エディターの統合。
### チャットインターフェイス
- チャット メッセージのスタイルが改善され、会話スレッド間の視覚的な階層と一貫性が向上しました。
- AI 生成の SQL と自然言語応答の信頼できる表示を保証するために、セッション処理とメッセージ レンダリング ロジックを修正しました。
