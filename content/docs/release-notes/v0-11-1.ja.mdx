# v0.11.1

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.11.1.md)

## 何が変わったのか
### 改善点
- 分析ロード中のレイアウトのシフトを防ぐために、SQL コンソール コパイロット パネルの [インサイト] タブの高さを安定させました。
- 一貫した多言語サポートを実現するために、分析関連のすべてのコピーを英語、スペイン語、日本語、中国語のロケールにローカライズしました。
### 修正
- インサイトが生成または更新されるときの結果テーブルと分析 UI コンポーネントのレンダリングに一貫性がない問題を解決しました。
