# v0.4.7

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.4.7.md)

## 何が変わったのか
### 修正
- デモ認証フローでの電子メール検証の失敗を修正しました。
- PostgreSQL および pglite のスキーマと移行の更新を含む、`better-auth `統合に関連するエラーを解決しました。
＃＃＃ 特徴
- 自動更新の信頼性を向上させるために、デスクトップ アプリに定期的な更新チェックを追加しました。
### 改善点
- すべての Web アプリケーションにわたって Next.js を v14.2.35 から v16.2.0 にアップグレードし、最新の React サーバー コンポーネント、ルーティングの改善、パフォーマンスの強化を可能にしました。
- KyselyORMを v0.28.12 から v0.28.14 に更新し、安定性とタイプ セーフティを若干改善しました。
