# v0.3.8

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.3.8.md)

## 何が変わったのか
### 修正
- 手動アプリケーション更新時のユーザー フィードバックの改善: 分類されたエラー ダイアログ (ネットワーク、サーバー、または汎用) を表示し、進行中の更新によってチェックがブロックされた場合にユーザーに通知し、サイレント ダウンロードを回避するために「後で通知する」設定を尊重します。
### ドキュメント
- エージェント関連の機能と使用法を文書化するために`AGENTS.md `を追加しました。
### 改善点
- データベース API (データベース、テーブル、ビュー、関数、マテリアライズド ビューなど) 全体での保守性と一貫性を向上させるために接続実装がリファクタリングされました。
- バックエンド ルート、ドライバー固有の実装 (ClickHouse)、フロントエンド権限ページなどの権限管理ロジックをリファクタリングし、正確性、拡張性、ロール/ユーザー スコープとの整合性を向上させました。
- 接続レジストリとルート処理をリファクタリングして機能ベースのアーキテクチャを採用し、よりクリーンなドライバーの抽象化と将来のデータベースのサポートを可能にします。
