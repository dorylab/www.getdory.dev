# v0.7.10

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.7.10.md)

## ¿Qué ha cambiado?
### Autenticación
- Se eliminó la funcionalidad de inicio de sesión de invitado e introdujo un nuevo flujo de inicio de sesión de demostración, que permite a los usuarios explorar rápidamente la aplicación sin crear una cuenta.
### SQL Mejoras en la experiencia de usuario de la consola
- Se mejoró la experiencia de Copilot en la consola SQL con un comportamiento refinado del modo SQL, una administración de pestañas más inteligente y una capacidad de respuesta mejorada del editor.
- Se actualizó el componente de la tabla de resultados para una mejor legibilidad, interacción y accesibilidad.
- Integración refinada del editor Monaco para una edición SQL más fluida, incluidas sugerencias de sintaxis y resaltado de errores.
### Interfaz de chat
- Estilo de mensajes de chat mejorado para una mejor jerarquía visual y coherencia en los hilos de conversación.
- Se corrigió la lógica de procesamiento de mensajes y manejo de sesiones para garantizar una visualización confiable de SQL generado por IA y respuestas en lenguaje natural.
