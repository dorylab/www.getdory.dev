# v0.12.2

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.12.2.md)

## 何が変わったのか
### オフラインデスクトップ認証
- ネットワーク依存性のない匿名サインイン フローを含む、デスクトップ アプリの完全なオフライン サポートを追加しました。
- Web セッション管理から切り離された、デスクトップ固有の認証セッション処理および回復ロジックの導入
- 個別のデスクトップ ログイン パスとランタイム エイリアスをサポートするために認証ルートと UI コンポーネントを更新しました
### 副操縦士の安定性の向上
- Copilot アクションの実行中に DuckDB 接続が誤ってアクセスされるという重大な問題を修正し、クラッシュを防止し、SQL 関連の AI 機能 (チャートの生成、クエリの書き換え、最適化など) の信頼性を向上させました。
### ユーザーの初期セットアップ
- 環境変数 (`INIT_USER_EM AI L `、` INIT_USER_PASSWORD `など) を使用して初期ユーザーに構成可能なブートストラップ ログインを追加し、セルフホストおよびローカル開発環境でのオンボーディングを簡素化しました。
### UI と UX の修正
- AI によって生成された分析と推奨事項が正確に表示されるように、SQL コンソールでの結果の洞察のレンダリングを修正しました。
### ドキュメントとツール
- 新しいセットアップ手順と、オフライン モードおよび初期ユーザー構成に関する説明を追加して` README.md `を更新しました
- 信頼性とエイリアス解決を向上させるために洗練されたスタンドアロン デスクトップ アプリ ビルド スクリプト (` build-app-standalone.sh `)
