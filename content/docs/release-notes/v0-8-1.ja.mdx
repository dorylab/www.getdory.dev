# v0.8.1

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.8.1.md)

## 何が変わったのか
### 改善点
- **チャットボットの国際化**: 英語、スペイン語、日本語、中国語のロケールの翻訳を更新し、チャットボット UI コンポーネントとインタラクション全体での多言語サポートを改善しました。
### 修正
- **SQL 生成の信頼性**:`generate-sql `クイック アクションを専用のサーバー側アクションにリファクタリングし、一貫性、テスト容易性、およびユーザーが自然言語プロンプトから SQL をリクエストしたときのエラー処理を改善しました。
