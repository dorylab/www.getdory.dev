# v0.21.1

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.21.1.md)

## What's Changed

### Fixes
- Fixed Mermaid diagram rendering issues in AI-generated responses.
- Corrected MCP request origin detection to ensure proper authentication and routing.
- Adjusted the Mermaid fullscreen dialog to properly fill the viewport for better usability.

### Improvements
- Upgraded Next.js to version 16.2.7 across all web-facing applications, improving stability and compatibility.

### Internal
- Updated Docker build configuration to include missing workspace manifests, ensuring consistent artifact packaging.
