# v0.14.1

[GitHub でソースを見る](https://github.com/dorylab/dory/blob/main/release-notes/v0.14.1.md)

## 変更点

### 修正
- アクションコンテキストに `schemaContext` がないため SQL Copilot のアクションが失敗する重大なエラーを修正し、AI 支援によるクエリ修正とインライン質問の信頼性を向上しました。

### 改善
- SQL Console の AI Fix ボタンに新しい `BorderBeam` ビジュアル効果を追加し、ユーザーへのフィードバックと一貫性を改善しました。
- `BorderBeamButton` を共有パッケージ `@dorylab/ui` に抽出し、アプリケーション間で再利用できるようにして UI コンポーネントの保守性を向上しました。

### ドキュメント
- `README.md` に説明と最新の利用ガイドを追加しました。
