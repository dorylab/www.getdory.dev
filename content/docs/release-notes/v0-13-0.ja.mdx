# v0.13.0

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.13.0.md)

## 何が変わったのか
＃＃＃ 特徴
- **MCP(モデル コンテキスト プロトコル) サーバー サポートの追加**: 新しいAPIエンドポイント (`/api/mcp/`、`/api/mcp/settings `、`/api/mcp/tokens `) と設定の UI コントロールにより、安全なエージェント アクセス、トークン管理、および MCP の統合が可能になります。
- **デスクトップ用の安定した MCP ポート**: デスクトップ アプリは、ローカル MCP プロキシに一貫した予測可能なポートを使用するようになり、信頼性とユーザー エクスペリエンスが向上しました。
### 改善点
- **強化された AI 分析機能**: AI 分析概要スキーマで数値フィールドが受け入れられるようになり、クエリ結果からより豊富なグラフ作成と洞察が可能になります。
- **洗練された SQL コンソールの洞察フロー**: 複数のデータベース ドライバーおよび自動化 API にわたる SQL コンソール機能 (エディター、サイドバー、テーブル ブラウザー統計、スキーマ イントロスペクションなど) の動作と応答性が向上しました。
### 修正
- **セキュリティ強化**: デスクトップのビルドおよび公証スクリプトのセキュリティ関連のエラーを解決し、脆弱性に対処するために依存関係の構成を更新しました。
- **Vercel 展開の修正**: Vercel 展開中のインストール エラーを解決するために Yarn 構成 (`.yarnrc.yml `、`.yarn/releases/`) を修正しました。
