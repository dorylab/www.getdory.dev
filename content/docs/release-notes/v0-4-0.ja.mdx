# v0.4.0

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.4.0.md)

## 何が変わったのか
＃＃＃ 特徴
- **PostgreSQL サポートの追加**: 基本接続、メタデータ検出、テーブル情報、およびスキーマの参照が PostgreSQL データベースで利用できるようになりました。
- **新しい Explorer インターフェイスの開始**: 統合されたドライバー対応データベース エクスプローラーが従来のカタログ ビューを置き換え、データベース、スキーマ、テーブル、ビュー、マテリアライズド ビュー、関数、および拡張機能の参照で PostgreSQL と ClickHouse をサポートします。
- **PostgreSQL 固有のテーブルに関する洞察**: インデックスの使用状況、テーブル サイズ、バキューム ステータスに関する PostgreSQL 統計カードをテーブル ブラウザに追加しました。
### 改善点
- **SQL コンソール サイドバーの強化**: マルチデータベース接続のサポートを改善するためにリファクタリングされ、i18n が改善され、接続切り替え時のリロードの問題が修正され、スクロールとスタイルが更新されました。
- **スキーマ サイドバーの拡張**: PostgreSQL をサポートし、更新された UI スタイルと一貫した国際化が含まれています。
- **テーブル ブラウザーの最新化**: ドライバーを認識するようにリファクタリングされ、データ プレビュー ロジックがドライバーに移動され、タブの構成と応答性が向上しました。
### 修正
- SQL エディターでの重複した提案の挿入を修正しました。
- SQL エディターで欠落していたスキーマの自動提案を修正しました。
- カタログおよびエクスプローラーの概要コンポーネントにおける i18n キーの不一致を修正しました。
- エクスプローラーのサイドバーのオブジェクト リスト (関数、スキーマ オブジェクトなど) にデータが表示されない問題を修正しました。
- エクスプローラーとアプリのサイドバーでのアクティブ状態の強調表示を修正しました。
