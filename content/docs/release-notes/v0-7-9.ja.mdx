# v0.7.9

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.7.9.md)

## 何が変わったのか
### 修正
- AI 支援 SQL 生成で複数のデータベース (MySQL + ClickHouse) を使用するときのプロンプト エラーを修正しました。
- チャットボットのレイアウトの問題を修正し、エージェント ツールの実行ステップの集計を改善して、より明確に見えるようにしました。
### 改善点
- SQL 安全性チェックが強化され、`PRAGMA `および` DESCRIBE `ステートメントを読み取り専用モードで使用できるようになり、SQLite および他の方言との幅広い互換性が可能になります。
- 簡素化され洗練された SQL 結果のレンダリング: 中間の再試行が非表示になり、折りたたみ動作がより直観的になり、結果の周囲の間隔が狭くなり、一貫性が高まりました。
- チャットボットのサイドバー UI を更新し、ナビゲーションとセッション管理を改善しました。
- SQL 関連のチャット インタラクションのフォールバック処理とメッセージ レンダリングが改善され、エラーと部分的な結果がよりわかりやすくなりました。
- ファイル拡張子を必要とせずにSSH秘密キーの選択を許可しました (例:` id_rsa.pem `の代わりに` id_rsa `という名前のキー)。
- チャット ツールから不要な背景スタイルを削除し、信頼性の高いポータブル デモのためにデモ SQLite データベース (`-shm `および`-wal `ファイルを含む) をバンドルしました。
