# v0.5.0

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.5.0.md)

## 何が変わったのか
### 新しいデータベースのサポート
- 接続フォーム、メタデータ探索、テーブル情報の取得、SQL 方言処理などの基本的な MySQL 接続サポートを追加しました。
- MariaDB 接続サポートを追加し、MySQL インフラストラクチャの多くを再利用しながら、MariaDB 固有の機能とエクスプローラー コンポーネントを提供しました。
### UIの改善
- 視覚的な一貫性とアクセシビリティを向上させるために、アプリのサイドバーのユーザー アバターからグレースケール フィルターを削除しました。
