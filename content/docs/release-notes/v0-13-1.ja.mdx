# v0.13.1

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.13.1.md)

## 何が変わったのか
### 改善点
- ビルド プロセスから Yarn リリースの依存関係を削除することで、Docker イメージのビルドが簡素化され、ビルド時間と複雑さが軽減されます。
### 修正
- Yarn リリース ファイルが最終的な Docker インストーラー イメージに正しく含まれていることを確認し、Yarn バイナリの欠落に関連する潜在的な実行時の問題を修正しました。
