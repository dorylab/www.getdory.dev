# v0.11.3

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.11.3.md)

## 何が変わったのか
### 修正
- スレッド表示中に予期しないエラーを引き起こす可能性があるチャットボット メッセージ レンダリング ロジックのエラーを修正しました。
- AI によって生成されたチャート結果におけるチャートのレンダリングの問題が解決され、信頼性と視覚的な一貫性が向上しました。
### 改善点
- パフォーマンスと更新された依存関係との互換性を向上させるために、チャート レンダリング コンポーネント (`chart-canvas.tsx `、` charts-result/index.tsx `) を更新しました。
- セッション処理、空の状態の表示、チャット入力など、複数のファイルにわたるチャットボット UI と動作が強化され、よりスムーズな対話と応答性の高いフィードバックが実現します。
- データベース クエリ支援の精度とコンテキスト認識を向上させるために、SQL 関連の AI プロンプトとツール (` sql.tools.ts `、` sql-runner.ts `) を洗練しました。
### 依存関係
- ` icu-minify `を 4.9.1 から 4.11.0 にアップグレードしました (推移的、` next-intl `経由)。
- ランタイムの安定性の向上とマイナーなバグ修正のために、` hono `を 4.12.14 から 4.12.18 にアップグレードしました。
- ローカリゼーション関連の修正と最適化を含め、` next-intl `を 4.9.1 から 4.9.2 にアップグレードしました。
