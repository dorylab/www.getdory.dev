# v0.12.0

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.12.0.md)

## What's Changed

### Features
- **Added DuckDB support**: Users can now connect to and query DuckDB databases directly from the web app, including connection setup, SQL execution, table browsing, and schema introspection.

### Dependencies
- Updated `fast-xml-builder` from `1.1.4` to `1.2.0`.
- Updated `fast-uri` from `3.1.0` to `3.1.2`.
