# v0.3.10

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.3.10.md)

## 何が変わったのか
＃＃＃ 特徴
- **保存されたクエリがフォルダーをサポートするようになりました**: 完全なCRUD操作、ドラッグ アンド ドロップによる並べ替え (フォルダーとクエリの両方)、およびネストされた検索を使用して、保存された SQL クエリをフォルダーに整理します。英語と中国語の i18n サポートが含まれます。
- **ドラッグ アンド ドロップによる並べ替え**: 視覚的なフィードバック、ドラッグ ハンドル、セクション認識境界を使用して、SQL コンソール サイドバー内のフォルダーとクエリを直感的に並べ替えます。
### 修正
- **PGLite スキーマ パリティ**:`saved_query_folders `テーブルを作成し、` folder_id `および` position `フィールドで` saved_queries `を更新するために欠落していた PGLite 移行を追加しました。これにより、ローカルの開発およびテスト環境が Postgres と一致するようになります。
### 改善点
- **ドキュメントの更新**:` README.md `を更新し、セットアップ手順と使用方法のガイダンスを更新しました。
