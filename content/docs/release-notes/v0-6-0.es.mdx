# v0.6.0

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.6.0.md)

## ¿Qué ha cambiado?
### Características
- Se agregó soporte SQLite experimental en toda la aplicación, que incluye:
  - Integración con la consola SQL y el navegador de tablas para bases de datos SQLite
  - Nuevos tipos de contexto Copilot diseñados para asistencia de consultas SQLite
  - Paquetes de aplicaciones de escritorio actualizados y flujos de trabajo de CI para incluir archivos binarios de SQLite
### Mantenimiento
- Dependencia indirecta actualizada `brace-expansion ` de `1.1.12` a `1.1.13`(parche de seguridad y estabilidad)
