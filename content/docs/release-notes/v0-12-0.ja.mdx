# v0.12.0

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.12.0.md)

## 何が変わったのか
＃＃＃ 特徴
- **DuckDB サポートの追加**: ユーザーは、接続セットアップ、SQL の実行、テーブルの参照、スキーマのイントロスペクションなど、Web アプリから直接 DuckDB データベースに接続してクエリを実行できるようになりました。
### 依存関係
- `fast-xml-builder `を`1.1.4`から`1.2.0`に更新しました。
- ` fast-uri `を`3.1.0`から`3.1.2`に更新しました。
