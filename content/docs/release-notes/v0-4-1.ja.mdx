# v0.4.1

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.4.1.md)

## 何が変わったのか
- **UI の一貫性**: PostgreSQL データベースに接続すると、ナビゲーション サイドバーとページ レイアウトの監視セクションと特権セクションが非表示になり、動作がバックエンド サポートの制限に合わせられます。
- **コードのクリーンアップ**: 監視ページと権限ページにわたる PostgreSQL 接続の冗長な接続準備チェックと関連する UI スキャフォールディングを削除しました。
