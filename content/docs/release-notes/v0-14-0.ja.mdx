# v0.14.0

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.14.0.md)

## 何が変わったのか

＃＃＃ 特徴
- **ローカル ファイル サポート**: Web アプリからローカル ファイル (Excel、CSVなど) に直接接続してクエリを実行するための完全な統合が追加されました。これには、ファイルのアップロード、スキーマ検査、テーブル プレビュー、SQL の実行が含まれます。
- **デスクトップ ファイル ピッカー**: シームレスなローカル ファイル オンボーディングのために、Electron デスクトップ アプリでネイティブ OS ファイルの選択を有効にしました。

### 改善点
- **UI の改良**: 接続ダイアログと SQL 結果テーブルの間隔と境界線が狭まり、視覚的な一貫性と明瞭さが向上しました。
- **ローカリゼーション**:MCP 設定、APIメッセージ、および接続 UI コンポーネントにわたる更新された翻訳 (英語、スペイン語、日本語、中国語) により i18n サポートが拡張されました。
- **Open Files UX**: データセットの分離、サイドバーのラベル付け、およびコミット メッセージ生成ロジックが改良され、ローカル ファイルを操作する際の発見性とコンテキスト認識が向上しました。

### 修正
- **Docker ビルドの信頼性**: イメージ ビルドの成功を妨げていた`Dockerfile `のエラーを修正しました。

### ドキュメント
- ` README.md `を更新し、セルフホスティング手順と完全なセルフホスティング ドキュメントにリンクする専用セクションを追加しました。

＃＃＃ 内部
- コンテナ公開を制御するための手動 Docker リリース GitHub アクション ワークフローを追加しました。
- 安定したエンドツーエンド CI テスト、特にデモ接続と SQL コンソール フロー。
