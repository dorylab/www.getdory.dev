# v0.4.1

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.4.1.md)

## ¿Qué ha cambiado?
- **Consistencia de la interfaz de usuario**: se ocultan las secciones Monitoreo y Privilegios en la barra lateral de navegación y los diseños de página cuando se conecta a una base de datos PostgreSQL, alineando el comportamiento con las limitaciones de soporte de backend.
- **Limpieza de código**: se eliminaron las comprobaciones redundantes de preparación de la conexión y la estructura de interfaz de usuario relacionada para las conexiones PostgreSQL en las páginas de supervisión y privilegios.
