# v0.12.1

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.12.1.md)

## ¿Qué ha cambiado?
### Características
- **Se agregó compatibilidad con DuckDB**: los usuarios ahora pueden conectarse y consultar bases de datos DuckDB directamente desde la aplicación web, incluida la configuración de la conexión, la integración de la consola SQL, la exploración de tablas y la introspección de esquemas.
### Correcciones
- **Se corrigieron errores de tiempo de ejecución de DuckDB**: se resolvieron problemas de inicialización y ejecución en la aplicación de escritorio para garantizar una conectividad estable de DuckDB y una ejecución de consultas.
- **Se corrigió el flujo de trabajo de la versión beta**: se corrigió un error de configuración en el flujo de trabajo de acciones de GitHub `beta.yml ` que impedía las compilaciones beta adecuadas.
