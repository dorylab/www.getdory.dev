# v0.5.5

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.5.5.md)

## 何が変わったのか
＃＃＃ 特徴
- SQL コンソール結果テーブルの仮想化テーブル (VTable) コンポーネントに自動列幅計算と自動調整動作が追加され、さまざまな列幅でのクエリ結果の読みやすさが向上しました。
### 修正
- VTable ヘッダー (`thead `) での水平スクロール (` scroll-x `) の失敗を修正し、すべてのテーブル セクションにわたって一貫したスクロール動作を保証します。
### 改善点
- SQL コンソール UI 内の視覚的な階層と一貫性を向上させるために、VTable コンポーネントのセクションの背景スタイルを更新しました。
