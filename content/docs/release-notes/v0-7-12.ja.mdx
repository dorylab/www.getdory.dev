# v0.7.12

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.7.12.md)

## 何が変わったのか
### 修正
- タブの切り替えまたは自動保存中に SQL エディターのコンテンツが失われることがある問題を修正しました。
- チャットボットおよび SQL コンソール コンテキストでの PostgreSQL データベースとスキーマ メタデータの誤った処理が解決され、クエリの精度とテーブルの提案が向上しました。
＃＃＃ 特徴
- データベースとスキーマのリストを手動で再ロードするための **更新ボタン** を SQL コンソールのサイドバーに追加しました。
- アクティブな接続タイプ (PostgreSQL、MySQL など) に適応する動的な SQL プロンプト生成 (`buildDialectSqlPrompt `) を導入し、AI によって生成されたクエリの関連性を向上させました。
### 改善点
- 列タグを遅延ロードすることでテーブル ブラウザのパフォーマンスを最適化し、大規模なスキーマの初期レンダリング時間を短縮しました。
- 一貫した` cursor-pointer `スタイルを使用して、複数のコンポーネント (ボタン、ドロップダウン、サイドバーなど) にわたる UI フィードバックを強化しました。
