# v0.5.2

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.5.2.md)

## 何が変わったのか
＃＃＃ 特徴
- SQL コンソール サイドバーに現在選択されているテーブルの視覚的な強調表示が追加され、ナビゲーションとコンテキスト認識が向上しました。
### 改善点
- SchemaSelect コンポーネントを更新して、スキーマを選択する際の明確さと使いやすさを向上させました。
- [データ] タブをテーブル ブラウザのデフォルト ビューとして設定し、テーブルの内容へのアクセスを合理化します。
- データベースの概要ページをリファクタリングおよび更新し、レイアウトと情報階層を改善しました。
### 修正とローカリゼーション
- 統計関連のコンポーネント (AI インサイト、インデックス使用状況、バキューム ステータス、ストレージの健全性など) を一貫したスタイルと更新されたコピーで更新しました。
- 国際化をサポートするために、複数の UI コンポーネントにわたって英語と中国語のローカライズ文字列を追加および更新しました。
