# v0.4.2

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.4.2.md)

## ¿Qué ha cambiado?
### Correcciones
- Se corrigieron fallas de compilación en las páginas de administración de privilegios (roles, usuarios y la vista principal de privilegios).
- Se ocultaron las secciones **Monitoreo** y **Privilegios** en la interfaz de usuario para conexiones PostgreSQL, donde estas funciones no son compatibles.
