# v0.8.2

## ¿Qué ha cambiado?
- **Se corrigió la visualización de mensajes de chat**: se resolvió un problema por el cual el mensaje "preguntar" del usuario no se mostraba correctamente en la interfaz de chat, lo que mejoraba la confiabilidad y claridad durante las conversaciones.
