# v0.4.8

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.4.8.md)

## ¿Qué ha cambiado?
### Características
- Se agregó soporte para un entorno beta dedicado, lo que permite realizar pruebas más seguras y obtener una vista previa de las próximas funciones.
### Mejoras
- Se eliminó la entrada de la organización de la barra lateral de navegación del usuario en la aplicación web, simplificando la interfaz de usuario para los usuarios dentro del contexto de una organización.
