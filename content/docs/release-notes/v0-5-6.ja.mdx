# v0.5.6

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.5.6.md)

## 何が変わったのか
＃＃＃ 特徴
- SQL コンソール結果テーブルに自動列幅調整 (autofit) が追加され、手動でサイズを変更しなくてもクエリ結果が読みやすくなりました。
### 修正
- 結果テーブルの`<thead>`の水平スクロール (` scroll-x `) の問題を修正し、スクロール中にヘッダーがデータ列と整列した状態を維持できるようにしました。
- アップデートのための正しいアセットの準備を妨げていた、macOS ビルド ワークフローのパッケージング エラーを解決しました。
### 改善点
- 結果テーブルコンポーネントのセクションの背景スタイルを更新することにより、視覚的な一貫性が強化されました。
- 列幅の計算ロジックが改善され、より正確で応答性の高いテーブル レイアウトが実現しました。
