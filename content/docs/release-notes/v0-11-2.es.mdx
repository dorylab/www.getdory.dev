# v0.11.2

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.11.2.md)

## ¿Qué ha cambiado?
### Mejoras
- Colores de fondo refinados en las pantallas de carga y de presentación tanto en aplicaciones web como de escritorio para mejorar la coherencia visual y la experiencia del usuario.
### Mantenimiento
- Se eliminaron los archivos de notas de la versión beta obsoletos (`v0.4.4-beta.0.md `,` v0.4.5-beta.0.md `,` v0.4.6-beta.0.md `) para agilizar la documentación.
