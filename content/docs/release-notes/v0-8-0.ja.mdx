# v0.8.0

[GitHub でソースを表示](https://github.com/dorylab/dory/blob/main/release-notes/v0.8.0.md)

## 何が変わったのか
＃＃＃ 特徴
- SQL 生成機能を SQL コンソールに追加しました。ユーザーは自然言語で質問し、対応する SQL クエリを生成してエディターに直接挿入できるようになりました。
- インライン質問の改善、コンテキスト認識の向上、および多言語ローカリゼーション (英語、スペイン語、日本語) のサポートにより、SQL コンソールの AI アシスタントが強化されました。
### 修正
- Next.js 設定の適切な読み込みを妨げていた`next.config.ts `の TypeScript タイプ エラーを修正しました。
### 改善点
- ` README.md `のドキュメントを更新し、説明と現在の使用方法のガイダンスを追加しました。
＃＃＃ メンテナンス
- アップグレードされた依存関係:
  - `@xmldom/xmldom ``0.8.12`から`0.8.13`へ
  - ` uuid `を`13.0.0`から`14.0.0`に (重大な変更; 最新のESM環境との互換性を確保)
