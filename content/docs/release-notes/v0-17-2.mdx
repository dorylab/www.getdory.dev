# v0.17.2

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.17.2.md)

## What's Changed

### Fixes
- Fixed chart rendering issues by removing nested responsive containers, improving layout stability and performance in chart result displays.
- Prevented infinite chart generation loops and skipped empty reasoning steps during AI chat responses, leading to more reliable and efficient chart creation.

### Improvements
- Added localized “save tips” UI guidance in the AI settings page to help users understand when and how configuration changes are persisted.
- Corrected multiple localization string inconsistencies and missing translations across English, Spanish, Japanese, and Chinese locales, improving UI text reliability and internationalization coverage.
