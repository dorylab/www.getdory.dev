# v0.4.6

[Ver fuente en GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.4.6.md)

## ¿Qué ha cambiado?
### Características
- **Soporte de organización**: se agregó funcionalidad completa para múltiples organizaciones, incluida la creación de organizaciones, configuraciones (detalles de la organización, miembros, invitaciones) y enrutamiento a través de segmentos dinámicos `[organization]`.
- **Manejo de vencimiento de sesión**: se introdujo la lógica de vencimiento de sesión en el sistema de autenticación para mejorar la seguridad y la administración de sesiones de usuario.
- **Integración de Dash**: se agregó soporte para Dash (probablemente refiriéndose a un panel o integración de análisis), incluida la lógica de autenticación del cliente actualizada, migraciones de bases de datos y cambios de esquema.
- **Integración de Sentinel**: Se agregó Sentinel (probablemente un servicio de monitoreo, observabilidad o seguridad) con las correspondientes actualizaciones de variables de entorno y enlaces de capa de autenticación.
### Mejoras
- **Configuración del entorno**:`.env.example ` actualizado con nuevas variables requeridas:`BETTER_AUTH_API_KEY ` y claves relacionadas con Sentinel.
