# v0.11.1

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.11.1.md)

## What's Changed

### Improvements
- Stabilized the height of the Insights tab in the SQL Console Copilot panel to prevent layout shifts during analysis loading.
- Localized all analysis-related copy across English, Spanish, Japanese, and Chinese locales for consistent multilingual support.

### Fixes
- Resolved inconsistent rendering of result table and analysis UI components when insights are generated or updated.

### Internal
- Updated dependency versions across `apps/admin`, `apps/electron`, and `apps/web` packages to align with the latest stable releases.
